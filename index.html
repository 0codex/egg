Welcome to Egg.

Egg is a hybrid mobile application designed to cover all the functionality exposed by Wordpress in its new JSON API v2: Posts, Posts Revisions, Pages, Media, Post Types, Post Statuses, Comments, Taxonomies, Categories, Tags and Users. Also implements most of the recommended plugins, allowing access to multi-site environment, themes, plugins, widgets, menus, sidebars ...

Egg in deep

Egg deep attempts to cover aspects of the world of Wordpress, not only in terms of management as a useful tool for webmaster, also implements components, plugins, themes and best practices for frontend development. Egg also is integrated with plugins as Buddypress or WooCommerce, to name a few examples, and describes a specification for creating wordpress themes compatible with hybrid mobile applications, enabling flexible and sustainable environment for wordpress developers.
